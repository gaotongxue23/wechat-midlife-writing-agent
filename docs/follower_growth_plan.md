# 公众号涨粉实验计划

## 当前问题

账号目前有 4 个粉丝。最新文章有 162 阅读，但新增关注为 0。现阶段不缺“泛泛共鸣”，缺的是读者关注这个账号的明确理由。

接下来 5 篇先不追求偶然爆款，重点验证三个指标：

- 阅读完成率：从 22% 提升到 30% 以上。
- 互动率：从 1.23% 提升到 2% 以上。
- 新增关注：每篇先争取至少 1 个，建立账号自己的转化基线。

这些是本账号的实验目标，不是行业平均值。

## 内容产品化

把文章稳定归入三个系列，让读者知道关注后会得到什么：

1. 子女边界课：金钱、带娃、同住、情绪控制和养老责任。
2. 晚年夫妻课：说话方式、沉默、嫌弃、照顾和旧账。
3. 养老底线课：养老金、住房、医疗、再婚和财产安排。

## 每篇固定结构

1. 前 120 字写一个具体场景，同时说清读者能得到什么。
2. 正文 1100-1500 字，最多 3 个分论点。
3. 每个分论点包含“生活场景 + 清醒判断 + 一个做法”。
4. 文末放一篇相关阅读。
5. 用一句栏目定位和下一篇预告邀请关注，避免只写“欢迎关注”。

## 每日复盘字段

- 阅读人数
- 平均阅读时长
- 阅读完成率
- 分享、收藏、推荐、点赞和留言
- 新增关注
- 阅读来源与曝光量（后台能看到时补充）

核心计算：

- 互动率 =（分享 + 收藏 + 推荐 + 点赞 + 留言）/ 阅读人数
- 关注转化率 = 新增关注 / 阅读人数

连续积累 5 篇后，再判断哪个系列真正能带来关注，不能只按阅读量选题。
