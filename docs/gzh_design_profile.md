# 半生以后清醒课：公众号视觉规范

本项目的文章写作完成后，统一调用 `gzh-design-skill` 生成公众号可粘贴 HTML。视觉目标不是“装饰更多”，而是让 50 岁以上读者看得清、读得稳，并形成稳定的账号辨识度。

## 默认主题

- 基础组件：`留白禅意风（zen-whitespace）`
- 项目视觉名：`清醒雅阅`
- 文章类型：生活 / 情感随笔、观点 / 深度随笔
- 使用方式：沿用留白禅意风的组件和骨架，只做中老年阅读友好型排版适配，不混用其他主题组件。

## 阅读友好参数

- 正文字号：17px，不使用 14px 或 15px 正文。
- 正文行高：2.0。
- 正文颜色：`#3F3F3F`，避免浅灰低对比度。
- 段落间距：24-28px。
- 章节标题：22-24px，字重 700。
- 每段只标记 1 个核心短语；长段最多 2 个，避免满篇下划线。
- 最强视觉锚点全文不超过 4 处。
- 一篇文章的卡片、引用块和提示块合计不超过 3 个。

## 色彩与气质

- 页面底色以白色为主，不做大面积米黄或深色背景。
- 主色使用低饱和墨绿，表达安定和清醒。
- 暗红只用于少量提醒或结尾关注引导，不用鲜红大色块。
- 不用紫色渐变、荧光色、复杂阴影、票据风、互联网产品风。
- 整体气质：温和、端正、有生活阅历，不老气，也不刻意年轻化。

## 图片规范

- 使用真实、自然的中国中老年生活场景。
- 人物表情克制，避免哭喊、冲突摆拍和过度磨皮。
- 室内环境朴素整洁，光线自然，保留生活细节。
- 封面主体放在中央安全区，兼顾横版封面与方形缩略图裁切。
- 正文图片宁少勿杂，通常 1-3 张；没有信息增量的装饰图不放。

## 固定排版结构

1. 开头引言卡：一个生活场景或一句核心判断。
2. 三点导读：只展示文章最重要的三个收获。
3. 正文：最多三个章节，自动编号 01 / 02 / 03。
4. 关键句：每节最多一个重点引用或结论块。
5. 相关阅读：放一篇同系列历史文章。
6. 结尾：账号定位、下一篇预告和温和关注引导，只出现一次。

## 每篇交付物

- `drafts/YYYY-MM-DD-主题.md`：原始成稿。
- `layouts/YYYY-MM-DD-主题_排版_清醒雅阅.html`：公众号干净正文。
- `layouts/YYYY-MM-DD-主题_排版_清醒雅阅_预览.html`：带“复制到公众号”按钮的预览页。
- `assets/covers/主题.png`：封面图。

HTML 必须通过 `gzh-design-skill/scripts/validate_gzh_html.py`，ERROR 与半角标点 WARNING 都清零后才能交付。
