# 内容规则沉淀

## 2026-07-09

最新文章《60岁以后，别把养老钱全交出去，哪怕是亲生子女》阅读 108、点赞 1、转发 2、收藏 0。

观察：

- 养老钱方向稳定但没有明显突破。
- 标题里有“亲生子女”的冲突感，但正文如果偏原则提醒，互动会偏弱。
- 历史上“晚年夫妻沉默”阅读 123，高于养老钱题，今天优先切回夫妻关系。

调整：

- 养老钱题隔几篇再写，不连续追同类选题。
- 优先做“晚年夫妻关系 + 具体生活场景 + 情绪冲突”的文章。
- 标题尽量包含“晚年夫妻 / 最怕的不是 X / 而是 Y”结构。

## 2026-07-10

文章《夫妻到了晚年，最怕的不是吵架，而是互相嫌弃》阅读 162、平均阅读 1.73 分钟、完成率 22%、分享 1、收藏 1、新增关注 0。

观察：

- 晚年夫妻冲突题比上一篇养老钱题的 108 阅读高 50%，选题方向有效。
- 原稿约 2790 字、五个分论点，后半部分观点重复，完成率只有 22%。
- 162 阅读没有新增关注，说明“有共鸣”不等于“值得关注”；账号需要提供连续、明确、可预期的价值。
- 互动率约 1.23%，下一步需要增加清单、话术和具体做法，提高收藏与转发理由。

调整：

- 接下来 5 篇控制在 1100-1500 字，最多 3 个分论点。
- 第一屏同时写冲突与读者收益，不做长铺垫。
- 每篇加入一个可记忆框架，例如“帮急不包办、亲近不越界、惦记不控制”。
- 文末固定写账号定位和下一篇具体预告，测试关注转化。
- 复盘优先级改为：新增关注 > 完成率 > 收藏/分享 > 阅读量。

## 2026-07-11

文章《人过60岁，越疼子女，越要守住这3条边界》阅读 106、分享 2、点赞 0。当前截图中，晚年夫妻文章为 164 阅读，养老钱文章为 109 阅读。

观察：

- 子女边界题回到约 100 阅读的基础盘，比晚年夫妻题低约 35.4%。
- 分享率约 1.89%，说明“三点边界”这种具体、可转述的结构仍值得保留。
- 当前没有完读率与新增关注，不能判断短稿是否改善留存和涨粉。

调整：

- 下一篇优先组合“晚年夫妻的高阅读兴趣 + 三点框架的分享动机”。
- 从下一篇开始，写作 Agent 与 `gzh-design-skill` 串联，固定输出精排 HTML 和一键复制预览页。
- 视觉采用“清醒雅阅”：17px 正文、2.0 行高、白底深灰、低饱和墨绿、少量暗红点睛。
- 排版以清晰、稳重和留白为先，不使用密集卡片、花哨渐变和过多强调。
- 连续观察 3 篇精排文章的完成率后，再判断设计是否有效。

新增观察：

- 1100-1500 字暂时保持，不继续盲目缩短。
- 平均阅读时长下降与篇幅缩短同时发生，应与完成率一起判断，不能单独判差。
- 0 新增关注说明下一阶段重点是账号定位、合集承接和下一篇预告，而不是继续优化读完率一个指标。
- 本篇尚未使用 `gzh-design-skill`，不能用这组数据评价“清醒雅阅”排版。

文章《夫妻到了晚年，再亲也要守住这3个分寸》有 275 位阅读者、平均阅读 1.52 分钟、完成率 35%、分享 3、点赞 1、收藏 1、新增关注 1。

- 这是目前阅读最高、也是首个新增关注为正的样本；“晚年夫妻 + 具体日常场景 + 三点可执行框架”继续作为优先组合。

- 下一篇换成“各自有生活，又彼此惦记”的温和共鸣角度，避免连续写“边界”，测试收藏和关注能否持续出现。
