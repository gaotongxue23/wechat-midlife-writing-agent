# WeChat Midlife Writing Agent

一个面向公众号「半生以后清醒课」的写作 Agent。

它服务于中老年读者，围绕晚年生活、夫妻关系、子女关系、养老钱、孤独感和家庭边界，帮助账号完成选题、评分、大纲、初稿、标题优化、编辑检查和发布后复盘。

## 适合的内容方向

- 60 岁以后，钱和尊严的关系
- 晚年夫妻之间的沉默、照顾、嫌弃和陪伴
- 子女长大后，父母如何放下期待
- 人到晚年，如何处理孤独感
- 养老不是靠嘴上孝顺，而是靠提前安排
- 老了以后，不要把全部情绪寄托在孩子身上
- 亲戚关系、邻里关系、人情往来的边界

## 仓库结构

```text
agent/
  system_prompt.md        # 可直接放进 Agent 的系统提示词
config/
  agent.config.json       # 机器可读的 Agent 配置
memory/
  articles.sample.json    # 历史文章数据样例
prompts/
  topic.md                # 选题提示词
  outline.md              # 大纲提示词
  draft.md                # 初稿提示词
  title.md                # 标题提示词
  retro.md                # 复盘提示词
```

## 使用方式

把 `agent/system_prompt.md` 作为 Agent 的系统提示词。

日常可以这样调用：

```text
帮我基于“晚年夫妻沉默”生成 10 个公众号选题。
```

```text
给这个标题打分：夫妻到了晚年，最怕的不是吵架，而是无话可说。
```

```text
按账号风格写一篇 1500 字公众号文章。
```

```text
这是昨天的数据：阅读104，点赞2，转发3，帮我复盘。
```

## 迭代原则

这个项目不追求一次写完。

后续每次使用时，应该把真实反馈沉淀回来：

- 哪类标题阅读更高
- 哪类主题转发更好
- 哪些开头留存更强
- 哪些表达显得太空或太鸡汤
- 哪些选题不适合这个账号

Agent 的真正资产不是提示词本身，而是持续积累的选题判断、标题经验和复盘数据。
